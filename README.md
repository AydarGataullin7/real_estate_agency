# Сайт риэлторского агентства

Сайт находится в разработке, поэтому доступна только страница со списком квартир и админка для наполнения БД.

## Реализованные функции

- Список квартир с фильтрацией по городу, цене и новостройкам
- Админка для управления квартирами, жалобами и собственниками
- Поиск по админке
- Лайки на квартиры
- Нормализация телефонных номеров через `django-phonenumber-field`
- Отдельная модель собственников (Owner) со связью многие-ко-многим с квартирами
- Инлайн-редактирование собственников на странице квартиры

## Запуск

1. Скачайте код
2. Установите зависимости командой:

```bash
  pip install -r requirements.txt
```

3. Создайте файл базы данных и примените все миграции:

```bash
python manage.py migrate
```

4. Запустите сервер:

```bash
python manage.py runserver
```

## Создание администратора

Для доступа к админке создайте суперпользователя:

```bash
python manage.py createsuperuser
```
